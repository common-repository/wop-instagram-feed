== Description ==
Using Shortcode, copy and paste this shortcode to view the feeds ` [wop_instagram_feed] `.
  
= Features =
* easy to install
* Easy to use with shortcode.

= Requirements =
This plugin requires PHP 5 or PHP 7 and WordPress 3.0 or greater. It works in standard WordPress environments. It is strongly recommended to use WordPress in the newest version.

== Installation ==

1. Go to Plugins > Add New. 
2. Searh "wop instagram feed"
3. install and activate the WordPress Plugin.
4. Add the shortcode `[wop_instagram_feed]` on your page.
5. go to WOP Instagram Menu add User ID and access token

== How to display WOP Instagram Feed ==

Using PHP Function, use this php function to call the feeds  

`<?php wop_instagram_feed(); ?> `

Using Shortcode, copy and paste this shortcode to view the feeds  

` [wop_instagram_feed] `
